# salm.dev

Personal website and blog built with Astro.

## Stack
- Astro
- React (for interactive components)
- Tailwind CSS
- MDX (for blog posts) - [Why MDX?](https://www.codemotion.com/magazine/frontend/how-to-create-an-mdx-blog-in-typescript-with-next-js/#:~:text=Typically%2C%20MDX%20is%20used%20for,to%20achieve%20with%20simple%20Markdown.)

## Development
```bash
# Install dependencies
npm install

# Start development server
npx astro dev

# Build for production
npx astro build

# Preview production build
npx astro preview
```

## Structure
- `/src/content/blog/` - Blog posts in MDX format
- `/src/pages/` - Main site pages
- `/src/components/` - Reusable components
- `/src/layouts/` - Page layouts
- `/src/styles/` - Global styles and fonts
